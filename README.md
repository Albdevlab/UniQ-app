# UNIQ

UNIQ is an educational and structural platform dedicated to the Qubic ecosystem.

## What UNIQ is
- A neutral information and learning hub
- A structured entry point for understanding Qubic
- A long-term educational initiative

## What UNIQ is NOT
- Not an investment platform
- Not a financial product
- Not a token sale or fundraising project

## Core principle
Observe → Learn → Understand → Optional Contribute

UNIQ does not provide financial advice and does not promote speculative behavior.
